# Канонический вид
Все поверхности второго порядка имеют канонический вид. Образуется он так: *все константы и/или переменные без квадратов уходят направо, переменные с квадратами - налево.* Причём, переменные с квадратами обязаны не иметь коэффициентов перед собой, а только под собой, поэтому организовываем [[Деление\|Деление]], если оно необходимо. Также может потребоваться [[Выделение полного квадрата\|Выделение полного квадрата]].
