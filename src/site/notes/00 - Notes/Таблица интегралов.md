# Таблица интегралов
## Правила интегрирования
$\int{{kf(x)}dx}=k\int{{f(x)}dx}$, где $k=const$
$\int{(f(x)\pm g(x))dx}=\int{f(x)dx}+\int{g(x)dx}$
$\int{udv}=uv-\int{vdu}$ - [[00 - Notes/Интегрирование по частям\|Интегрирование по частям]]
## Простейшие интегралы
$\int{dx}=x+C$
$\int{{x^{n}}dx}=\frac{x^{n+1}}{n+1}+C,n\neq-1$
$\int{{e^{x}}dx}=e^{x}+C$;
- В общем виде: $\int{\frac{dx}{e^{kx}}}=-\frac{1}{ke^{kx}}+C$
$\int{{a^{x}}dx}=\frac{a^{x}}{\ln{a}}+C$
$\int{{\frac{dx}{x}}}=\ln{|x|}+C$

$\int{{\sin{x}}dx}=-\cos{x}+C$
$\int{{\cos{x}}dx}=\sin{x}+C$
$\int{{\frac{dx}{\sin^{2}{x}}}}=-\cot{x}+C$
$\int{\frac{dx}{\cos^{2}{x}}}=\tan{x}+C$

$\int{\frac{dx}{x^{2}-a^{2}}}=\frac{1}{2a}\ln{\left|\frac{x-a}{x+a}\right|}+C$;
- **в частности** $\int{\frac{dx}{x^{2}-1}=\frac{1}{2}\ln{\left|\frac{x-1}{x+1}\right|}}$
$\int{\frac{dx}{x^{2}+a^{2}}}=\frac{1}{a}\arctan{\frac{x}{a}}+C$;
- **в частности** $\int{\frac{dx}{x^{2}+1}}=\arctan{x}+C$
$\int{\frac{dx}{\sqrt{x^{2}\pm A}{}}}=\ln{|x+\sqrt{x^{2}\pm A}{}|}+C$
$\int{{\frac{1}{\sqrt{a^{2}-x^{2}}{}}}dx}=\arcsin{\frac{x}{a}}+C=-\arccos{\frac{x}{a}}+C$

$\int{\sinh{x}dx}=\cosh{x}+C$
$\int{\cosh{x}dx}=\sinh{x}+C$
$\int{\frac{dx}{\sinh^{2}{x}}}=-\coth{x}+C$
$\int{\frac{dx}{\cosh^{2}{x}}}=\tanh{x}+C$
## Дополнительные интегралы
$\int{\frac{dx}{x^{2}}=-\frac{1}{x}+C}$
$\int{\frac{dx}{\sqrt{x}{}}}=2\sqrt{x}{}+C$

$\int{\frac{dx}{e^{x}}}=\int{e^{-x}dx}=-\frac{1}{e^{x}}+C$

$\int{\ln{x}dx}=x(\ln{x}-1)+C$
$\int{x\ln{x}dx}=\frac{x^{2}}{2}(\ln{(x)-\frac{1}{2}})+C$
$\int{\ln{\sqrt{x}}dx}=\frac{x}{2}(\ln{x}-1)+C$
$\int{x\ln{\sqrt{x}}dx}=\frac{x^{2}}{4}\left( \ln{x}-\frac{1}{2} \right)+C$

$\int{\sqrt{x^{2}\pm a}\space dx}=\frac{x}{2}\sqrt{x^{2}\pm a}+\frac{a}{2}\ln{|x+\sqrt{x^{2}\pm a}|}+C$
$\int{\sqrt{a^{2}- x^{2}}\space dx}=\frac{x}{2}\sqrt{a^{2}- x^{2}}+\frac{a^{2}}{2}\arcsin{\frac{x}{a}}+C$

$\int{\tan{x}dx}=-\ln{|\cos{x}|}+C$
$\int{\cot{x}dx}=\ln{|\sin{x}|}+C$

$\int{\sin^{2}{x}dx}=\frac{x}{2}-\frac{\sin{2x}}{4}+C$
$\int{\cos^2{x}dx}=\frac{x}{2}+\frac{\sin{2x}}{4}+C$

$\int{\arcsin{x}dx}=x\,\arcsin\left(x\right)+\sqrt{1-{x}^{2}}+C$
$\int{\arccos{x}dx}=x\,\arccos\left(x\right)-\sqrt{1-{x}^{2}}+C$
$\int{\arctan{x}dx}=x\,\operatorname{arctg}\left(x\right)-\dfrac{\ln\left({x}^{2}+1\right)}{2}+C$
$\int{\operatorname{arccot}{x}dx}=x\,\operatorname{arcctg}\left(x\right)+\dfrac{\ln\left({x}^{2}+1\right)}{2}+C$

$\int{\frac{dx}{\operatorname{sh}{x}}=\frac{1}{2}\ln{\left| \frac{\operatorname{ch}{x}-1}{\operatorname{ch}{x}+1} \right|}}+C$
$\int{\frac{dx}{\operatorname{ch}{x}}}=\arctan{\operatorname{sh}{x}}+C$
$\int{\frac{dx}{\operatorname{th}{x}}}=\ln{|\operatorname{sh}{x}|}+C$
$\int{\frac{dx}{\operatorname{cth}{x}}}=\ln{\operatorname{ch}{x}}+C$
