# Ряд Фурье
>[!info]- Предпосылки

## Определение
Ряд Фурье - [[00 - Notes/Разложение функции в ряд\|разложение]] [[00 - Notes/Функция (Алгебра)\|функции]] $f(x)$ в [[00 - Notes/Функциональный ряд\|Функциональный ряд]]. Если $f(x)$ определена на отрезке $[-\pi;\pi]$, то ее можно разложить в Ряд Фурье по специальной формуле (приведена ниже).
## Теорема Дирихле о разложимости функции в ряд Фурье
>[!warning] Не путать с [[Признак Дирихле о сходимости числовых рядов\|признаком Дирихле о сходимости числовых рядов]]

>[!theorem]- Формулировка
>Пусть $f(x)$ - [[Периодическая функция\|периодическая функция]] с [[Период\|периодом]] $2\pi$. Пусть на интервале $[-\pi;\pi]$ эта функция имеет *конечное* количество [[Экстремум\|точек экстремума]] и может иметь *конечное* количество [[Разрыв\|точек разрыва]], причем *только I рода* (без бесконечностей), тогда такая функция разлагается в ряд Фурье:
> $$\frac{f(x-0)+f(x+0)}{2}=\frac{a_{0}}{2}+\sum\limits_{n=1}^{\infty}{(a_{n}\cdot \cos{(nx)}+b_{n}\cdot \sin{(nx)})}$$
## Формулы
>[!tip] Готовые интегралы
>Интегралы ниже возникают при нахождении $a_{n}$ и $b_{n}$
>
>| $f(x)$      | $F(x)$                                                                       |
>| ----------- | ---------------------------------------------------------------------------- |
>| $x\sin{nx}$ | $\dfrac{\sin\left(n\,x\right)}{{n}^{2}}-\dfrac{x\,\cos\left(n\,x\right)}{n}$ |
>| $x\cos{nx}$ | $\dfrac{x\,\sin\left(n\,x\right)}{n}+\dfrac{\cos\left(n\,x\right)}{{n}^{2}}$ |
### Значение ряда Фурье на границах периода
>[!error] Логика ниже не применима для самой функции $f(x)$, потому что в упражнениях по условию функция не определена в точках на конце периода.

Если у нас период $T=2l$ ($l \in\mathbb{R}$), то в точках $\pm l$, $\pm 2l$, $\pm 3l$ и т.д. значение функции $S(x)$ будет равно среднему арифметическому между пределом $f(x)$ в левой границе периода и пределом $f(x)$ в правой границе периода. Или проще:$$S(-l)=S(l)=\frac{S(-l+0)+S(l+0)}{2}$$

### Для любых функций

| Коэффициент/Период | $$T=2\pi \ (x \in[-\pi;\pi])$$                                                                    | $$T=2l \ (x \in[-l;l])$$                                                                                                      | $$T=b-a,l=\frac{b-a}{2}\ (x \in[a;b])$$                                                                                       |
| ------------------ | ------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------- |
| $$f(x)$$           | $$f(x)\textasciitilde \frac{a_{0}}{2}+\sum\limits_{n=1}^{\infty}{(a_{n}\cos{nx}+b_{n}\sin{nx})}$$ | $$f(x)\textasciitilde \frac{a_{0}}{2}+\sum\limits_{n=1}^{\infty}{(a_{n}\cos{\frac{\pi nx}{l}}+b_{n}\sin{\frac{\pi nx}{l}})}$$ | $$f(x)\textasciitilde \frac{a_{0}}{2}+\sum\limits_{n=1}^{\infty}{(a_{n}\cos{\frac{\pi nx}{l}}+b_{n}\sin{\frac{\pi nx}{l}})}$$ |
| $$a_{0}$$          | $$a_{0}=\frac{1}{\pi}\int\limits_{-\pi}^{\pi}{f(x)dx}$$                                           | $$a_{0}=\frac{1}{l}\int\limits_{-l}^{l}{f(x)dx}$$                                                                             | $$a_{0}=\frac{1}{l}\int\limits_{a}^{b}{f(x)dx}$$                                                                              |
| $$a_{n}$$          | $$a_{n}=\frac{1}{\pi}\int\limits_{-\pi}^{\pi}{f(x)\cos{nx}dx}$$                                   | $$a_{n}=\frac{1}{l}\int\limits_{-l}^{l}{f(x)\cos{\frac{\pi nx}{l}}dx}$$                                                       | $$a_{n}=\frac{1}{l}\int\limits_{a}^{b}{f(x)\cos{\frac{\pi nx}{l}}dx}$$                                                        |
| $$b_{n}$$          | $$b_{n}=\frac{1}{\pi}\int\limits_{-\pi}^{\pi}{f(x)\sin{nx}dx}$$                                   | $$b_{n}=\frac{1}{l}\int\limits_{-l}^{l}{f(x)\sin{\frac{\pi nx}{l}}dx}$$                                                       | $$b_{n}=\frac{1}{l}\int\limits_{a}^{b}{f(x)\sin{\frac{\pi nx}{l}}dx}$$                                                        |
### Для четных функций
| Коэффициент/Период | $T=2\pi \ (x \in[-\pi;\pi])$                                                                     | $T=2l \ (x \in[-l;l])$                                                                                         |
| ------------------ | ------------------------------------------------------------------------------------------------ | -------------------------------------------------------------------------------------------------------------- |
| $f(x)$             | $$f(x)\textasciitilde \frac{a_{0}}{2}+\sum\limits_{n=1}^{\infty}{\left( a_{n}\cos{nx} \right)}$$ | $$f(x)\textasciitilde \frac{a_{0}}{2}+\sum\limits_{n=1}^{\infty}{\left( a_{n}\cos{\frac{\pi nx}{l}} \right)}$$ |
| $a_{0}$            | $$a_{0}=\frac{2}{\pi}\int\limits_{0}^{\pi}{f(x)dx}$$                                             | $$a_{0}=\frac{2}{l}\int\limits_{0}^{l}{f(x)dx}$$                                                               |
| $a_{n}$            | $$a_{n}=\frac{2}{\pi}\int\limits_{0}^{\pi}{f(x)\cos{nx}dx}$$                                     | $$a_{n}=\frac{2}{l}\int\limits_{0}^{l}{f(x)\cos{\frac{\pi nx}{l}}dx}$$                                         |
### Для нечетных функций
| Коэффициент/Период | $T=2\pi \ (x \in[-\pi;\pi])$                                 | $T=2l \ (x \in[-l;l])$                                                 |
| ------------------ | ------------------------------------------------------------ | ---------------------------------------------------------------------- |
| $f(x)$             | $$\sum\limits_{n=1}^{\infty}{b_{n}\sin{nx}}$$                | $$\sum\limits_{n=1}^{\infty}{b_{n}\sin{\frac{\pi nx}{l}}}$$            |
| $b_{n}$            | $$b_{n}=\frac{2}{\pi}\int\limits_{0}^{\pi}{f(x)\sin{nx}dx}$$ | $$b_{n}=\frac{2}{l}\int\limits_{0}^{l}{f(x)\sin{\frac{\pi nx}{l}}dx}$$ |
### Разложение непереодических функций
В общем случае, чтобы разложить непереодическую функцию $f(x)$ в ряд Фурье, нужно доопределить её до периодической относительно $x=0$. Если функция определена на $[0;L]$, то её нужно доопределить до интервала $[-L;L].$
### Разложение только по синусам или косинусам
>[!tldr] Разложение по косинусам эквивалентно разложению четной функции, разложение по синусам эквивалентно разложению нечетной функции

>[!info]- Разложение по синусам
>Чтобы разложить непереодическую $f(x)$ функцию по синусам на интервале $[0;L]$, нужно доопределить функцию на промежутке $[-L;0]$ так, что $f(-x)=-f(x)$ (то есть до *нечетной функции*), и разложить её в ряд Фурье на интервале $[-L;L]$ с периодом $T=2L$, пользуясь формулами для нечетных функций

>[!info]- Разложение по косинусам
>Чтобы разложить непереодическую $f(x)$ функцию по косинусам на интервале $[0;L]$, нужно доопределить функцию на промежутке $[-L;0]$ так, что $f(-x)=f(x)$ (то есть до *четной функции*), и разложить её в ряд Фурье на интервале $[-L;L]$ с периодом $T=2L$, пользуясь формулами для четных функций
