| [[00 - Notes/Окружность\|Окружность]]           | $x^{2}+y^{2}=r^{2}$                         | -                    | $F=(\pm r;0)$                     | 0                                        | 0                         | -                    | $x=\pm \frac{r}{\varepsilon}$                     |
| [[Гипербола\|Гипербола]]            | $\frac{x^{2}}{a^{2}}-\frac{y^{2}}{b^{2}}=1$ | $a>b$                | $F=(\pm c;0)$                     | $c=\sqrt{a^{2}+b^{2}}$                   | $\varepsilon=\frac{c}{a}$ | $y=\pm \frac{b}{a}x$ | $x=\pm \frac{a}{\varepsilon}$ (если не повернута) |
| Равносторонняя гипербола | $x^{2}-y^{2}=a^{2}$                         | -                    | $F=(\pm a;0)$                     | $c=a\sqrt{2}$                            | $\sqrt{2}$                | $y=\pm x$            | $x=\pm \frac{a}{\sqrt{2}}$                        |
| [[Парабола\|Парабола]]             | $y^{2}=2px;p>0$                             | -                    | $F\left(\frac{p}{2};0\right)$[^1] | -                                        | 1                         | -                    | $x+\frac{p}{2}=0$                                 |

| Название  | [[Параллельный перенос\|Параллельный перенос]]                            | Поворот на $90^{o}$ | Поворот на $180^o$ |
| --------- | --------------------------------------------------- | ------------------- | ------------------ |
| Эллипс    | $\frac{(x-x_{0})^{2}}{a}+\frac{(y-y_{0})^{2}}{b}=1$ | $a<b$               | -                  |
| Гипербола | $\frac{(x-x_{0})^{2}}{a}-\frac{(y-y_{0})^{2}}{b}=1$ | Справа $-1$         | -                  |
| Парабола  | $(y-y_{0})^{2}=2p(x-x_{0})$                         | $x^2=2py$           | $-y^{2}=2px$       |

[^1]: Константа $p$ называется *фокальным параметром* и показывает расстояние между директрисой и фокусом.
