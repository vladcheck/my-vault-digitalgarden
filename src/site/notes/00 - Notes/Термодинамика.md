# Молекулы
[[Молекула\|Молекула]] (латинское «массочка») – электрически нейтральная частица, построенная из двух и более атомов, связанных ковалентными связями. При классическом рассмотрении молекул, атомы составляющие молекулы возможно считать материальными точками
## [[Степени свободы\|Степени свободы]]
Степени свободы системы – независимые переменные, полностью определяющие положение выбранной системы в пространстве.
- Одноатомные молекулы (3 степени свободы) – молекулы инертных газов – $\text{He, Ne, Ar, Kr, Xe, Rn}$.
- Двухатомные молекулы (5 степеней свободы)
- Трехатомные и более крупные молекулы (6 степеней свободы)
# Полная энергия
$<E_{полн}> =\frac{i}{2}kT$
# [[Внутренняя энергия\|Внутренняя энергия]]
$i=n_{пост}+n_{вращ}+2n_{кол}$
$U=N \frac{i}{2}kT=\frac{\nu i}{2}RT$
### Колебательные степени свободы
$n_{кол}=3N-n_{пост} - n_{вращ}$, где $N$ - количество атомов
# Моли, [[Молярная масса\|Молярная масса]], количество вещества
В 1 моли вещества содержится 12 грамм углерода $^{12}C$
$N_{A}$ - количество молекул в одной моли вещества
$\mu=\frac{N}{N_{A}}=\frac{m}{M}$, где $N$ - количество молекул вещества, $M$ - молярная масса
# [[Идеальный газ\|Идеальный газ]]
1. Молекулы идеального газа считаются материальными точками.
2. Молекулы идеального газа не взаимодействуют друг с другом (т.е. внутренняя энергия идеального газа состоит только из кинетической энергии теплового движения молекул)
3. Молекулы идеального газа абсолютно упруго взаимодействуют со стенками сосуда в котором находятся.
$P=nkT$, где $n$ - концентрация: $n=\frac{N}{V}\left[ \frac{\text{молекул}}{м^{3}} \right]$; $k = 1,38\cdot10^{-23} \left[\frac{\text{Дж}}{\text{К}}\right]$ 
$PV=\mu RT$ - уравнение Менделеева-Клапейрона, уравнение состояния ИГ ($R=8,31 \frac{Дж}{моль\cdot К}$)
# Процессы в идеальном газе
Изопроцесс – такой процесс, протекающий с системой постоянной массы, при котором один из параметров состояния ($P, V, T$) фиксирован.
- Изобарный - постоянное давление - $\frac{V_{1}}{T_{1}}=\frac{V_{2}}{P_{2}}=const$ - Закон Гей-Люссака
- Изохорный - постоянный объем - $\frac{P}{T}=\frac{\mu R}{V}=const$ - Закон Шарля
- Изотермический - постоянная температура - $P_{1}V_{1}=P_{2}V_{2}=const$ - Закон Бойля-Мариотта

# Газовые законы
## [[Закон Дальтона\|Закон Дальтона]]
Давление идеального газа, состоящего из других, не взаимодействующих химически друг с другом идеальных газов, равно сумме давлений этих газов:
$$P=\sum_{i=1}^{n}{p_{i}}$$
## [[Закон Авогадро\|Закон Авогадро]]
$газ_{1}\neq газ_{2}:T_{1}=T_{2},P_{1}=P_{2}\to N_{1}=N_{2}$
