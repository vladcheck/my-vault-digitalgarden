# КПД
## Определение
Коэффициент отдачи механизмом полезной^[1] энергии называется КПД.

## Формула
$$\mu=\frac{N_{производимая}}{N_{потребляемая}}\Leftrightarrow 1-\mu=\frac{N_{потребляемая}-N_{производимая}}{N_{потребляемая}}$$
---
[1]: т.е. той, которая нам нужна для совершения работы.
